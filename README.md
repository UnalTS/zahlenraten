# Zahlenraten

Ein kleines, anfaengerfreundliches Browser-Spiel mit HTML, CSS und JavaScript.

## Beschreibung

Der Computer waehlt zufaellig eine Zahl zwischen 1 und 100.  
Du gibst Tipps ein und bekommst direkt Feedback:

- `Zu niedrig!` mit Liste der zu niedrigen Zahlen (`↑`)
- `Zu hoch!` mit Liste der zu hohen Zahlen (`↓`)
- Treffer mit Anzeige der benoetigten Versuche

## Features

- Eingabe per Button oder `Enter`
- Zaehlt die Anzahl der Versuche
- Getrennte Historie fuer zu niedrige und zu hohe Tipps
- Reset-Button fuer ein neues Spiel
- Einfache, klare UI

## Projektstruktur

```text
zahlenraten/

```

## Starten

1. Repository klonen oder herunterladen.
2. Datei `index.html` im Browser oeffnen.

Keine Installation und keine zusaetzlichen Pakete notwendig.

## Technologien

- HTML5
- CSS3
- JavaScript (Vanilla JS)
